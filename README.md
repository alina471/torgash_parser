# torgash_parser

# Парсер лотов с сайта torgi.org

Этот проект предназначен для извлечения данных о лотах (название, цена, ссылка)  
из заранее сохранённой HTML-страницы сайта **torgi.org**, раздела открытых торгов:

https://torgi.org/index.php?class=Auction&action=List&mod=Open&AuctionType=All

Программа обрабатывает сохранённую HTML-страницу, парсит её с помощью BeautifulSoup  
и выводит лоты, отсортированные по цене (от дорогих к дешёвым).  
Пользователь может указать минимальную и максимальную цену, и программа отфильтрует результаты.

---

## Функциональность

- Загружает локальный HTML-файл (например, `torgi_page.html`)
- Извлекает:
  - **название лота**
  - **цену** (нормализует строку в целое число)
  - **ссылку** на страницу лота
- Позволяет вводить **минимальную и максимальную цену**
- Показывает только подходящие лоты
- Сортирует результат по убыванию цены
- Сохраняет результат в файл **trades.json**

---

## Подготовка HTML-файла

Так как сайт подгружает данные динамически, необходимо сохранить страницу вручную:

1. Открыть страницу торгов в браузере  
   https://torgi.org/index.php?class=Auction&action=List&mod=Open&AuctionType=All

2. Прокрутить страницу до конца, дождаться полной загрузки всех лотов  
3. Нажать: **Ctrl+S → Сохранить как → "Веб-страница полностью"**  
4. Сохранить файл, например: `torgi_page.html`  
5. Поместить файл рядом с `parse_trades.py`
